Description

Transportation is a non-separable part of any society. Transportation has always played an important role in influencing the formation of urban societies. But sometimes transportation is held mutely in fewer situations like school/ college transportation or public buses. This project aims to increase the communication between the on-road vehicles and the end users (Students/Parents/Locals). The smart vehicle tracker based on IoT technology is developed using Esp12F, GSM and GPS modules and Raspberry pi zero. To detect the position of the vehicle, a GPS module is used. Also, to transmit all the data with the coordinates of the vehicle, the ESP WI-FI board is integrated with the system along with a GSM module for a direct internet connection. The server is built using Raspberry pi zero and Flask framework. A WhatsApp bot is also integrated into the server. Any end-user from around the world can access the current location of the vehicle using the internet via the website or the WhatsApp bot.
